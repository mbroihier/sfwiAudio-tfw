Test Case FINISH-0010
Title When a podcast completes, the main menu is presented
Objective

This test case is intended to verify the following implied or specified requirements:

  • When the podcast completes, the menu will revert to the main/initial menu.

Letting the podcast play to completion should cause the app to revert to the initial display menu.
Setup
Actions
  1. From the main menu, select a podcast.
  2. Let the podcast play to completion.
  3. Verify the play menu is exited and the app returns to the main (selection) menu.
Expected When the podcast completes, control will return to the main menu.
Results PASSED
